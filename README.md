# ImaGestures
### In this Application Used Various Gestures on Image View.
## Elements:-
* Tap Gesture - Tap on Image View that time open ImagePicker View.
* Pinch Gesture
* Rotate Gesture
* Swipe Gesture
* Pan Gesture
